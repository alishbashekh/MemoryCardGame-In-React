# Memory Card Matching Game
A fun and interactive Memory Card Matching Game built using ReactJS and Tailwind CSS.

## Features
-> Easy, Medium, Hard levels
-> Card flip animation
-> Move counter and best score
-> Sound effects with toggle
-> Responsive design

## How The Game Works
1. Click two cards
2. If they match, they stay open
3. If not, they flip back after 1 second
4. Move counter increases
5. Best score is saved in localStorage
